# Creating a Virtual Environment in backend_ppe

## ➤ Step 1: Check Python Version
Ensure that Python 3.11 is installed on your system.
## ➤ Step 2: Install the required python version
Install Python 3.11 (If Not Available)
If Python 3.11 is not installed, download it from the Microsoft Store or the official Python website.
## ➤ Step 3: Creation and activation of venv, commands
Create and Activate the Virtual Environment
<pre>py -3.11 -m venv venv 
.\venv\Scripts\Activate.ps1 
pip install -r requirements.txt 
python main.py </pre>
